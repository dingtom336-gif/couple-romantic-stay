# couple-romantic-stay

## Overview

Find romantic hotels for couples — king-size beds, scenic views, intimate atmosphere, and special couple amenities like champagne and spa packages. Wraps the flyai-cli to provide real-time travel data with booking links.

## Quick Start

### Install Skill

```bash
clawhub install couple-romantic-stay
# or
npx skills add alibaba-flyai/flyai-skill/tree/main/skills/couple-romantic-stay
```

### Install CLI

```bash
npm i -g @fly-ai/flyai-cli
```

### Verify

```bash
flyai --version
```
